# Ammu B Portfolio

A simple responsive portfolio website built with HTML, CSS, and JavaScript.

## Pages

- Home
- About
- Skills
- Projects
- Contact

## Features

- Mobile responsive layout
- Hash-based routing
- Mobile navigation menu
- Resume-based portfolio content
- Local profile image

## Files

- `index.html` - Main website structure
- `styles.css` - Website styling and responsive design
- `script.js` - Simple route switching and mobile menu behavior
- `assets/ammu-b-profile.jpeg` - Portfolio profile image

## How to Run

Open `index.html` directly in a web browser.

You can navigate using:

- `#/home`
- `#/about`
- `#/skills`
- `#/projects`
- `#/contact`
